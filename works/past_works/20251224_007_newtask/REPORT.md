# 钼酸根与硫代钼酸根水解平衡问题求解报告

## 问题描述

给定反应控制了稀薄水溶液中钼酸根与硫代钼酸根的逐步水解与硫取代序列：
- MoS₄²⁻ + H₂O(l) ⇌ MoOS₃²⁻ + H₂S(aq)        K₁ = 1.3 × 10⁻⁵
- MoOS₃²⁻ + H₂O(l) ⇌ MoO₂S₂²⁻ + H₂S(aq)      K₂ = 1.0 × 10⁻⁵
- MoO₂S₂²⁻ + H₂O(l) ⇌ MoO₃S²⁻ + H₂S(aq)      K₃ = 1.6 × 10⁻⁵
- MoO₃S²⁻ + H₂O(l) ⇌ MoO₄²⁻ + H₂S(aq)        K₄ = 6.5 × 10⁻⁶

初始条件：
- [MoS₄²⁻]₀ = 2.00 × 10⁻⁷ mol/L
- pH = 5.00
- V_aq = 1.00 L, V_g = 5.00 L
- T = 298.15 K
- H₂S的亨利常数 H = 9.741 × 10⁻³ atm/(mol·m⁻³)

## 求解过程

我们建立了包含以下平衡的方程组：
1. 四个水解平衡方程
2. 钼质量守恒方程
3. 气液平衡方程（H₂S挥发平衡）

使用迭代法求解方程组，考虑到平衡常数较小，反应进行程度较低。

## 计算结果

### 各物种平衡浓度（mol/L）
- [MoS₄²⁻] = 2.00e-07
- [MoOS₃²⁻] = 4.81e-59
- [MoO₂S₂²⁻] = 1.78e-97
- [MoO₃S²⁻] = 1.05e-135
- [MoO₄²⁻] = 2.53e-174
- [H₂S(aq)] = 4.81e-59
- [HS⁻(aq)] = 6.34e-61
- [S²⁻(aq)] = 7.98e-69
- H₂S(g)分压 = 1.15e-62 atm

### 物料平衡验证
- 钼质量守恒符合
- 气液平衡符合

## 最终答案

### 归一化浓度/气压
- 含钼物种：MoS₄²⁻: 2.00e-07, MoOS₃²⁻: 4.81e-59, MoO₂S₂²⁻: 1.78e-97, MoO₃S²⁻: 1.05e-135, MoO₄²⁻: 2.53e-174
- 含硫不含钼物种：H₂S(aq): 4.81e-59, HS⁻(aq): 6.34e-61, S²⁻(aq): 7.98e-69, H₂S(g): 1.15e-62

### 浓度/气压乘积比值
- 含钼物种浓度乘积: 4.74e-442 (接近于0的极小值)
- 含硫不含钼物种浓度/气压乘积: 2.79e-249
- **浓度/气压乘积比值: 1.69e-193**

## 结论

在给定条件下，钼酸根与硫代钼酸根的水解反应几乎不发生，绝大多数钼仍以MoS₄²⁻形式存在。这主要是因为水解反应的平衡常数相对较小，在pH=5.00的条件下，反应向右进行的程度非常有限。H₂S由于挥发性强，主要以气相形式存在于气相中。
