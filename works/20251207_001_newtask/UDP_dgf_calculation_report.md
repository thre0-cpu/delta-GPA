# UDP在特定条件下的生成自由能计算结果报告

## 计算任务
- 计算UDP（尿苷二磷酸）在特定条件下的生成自由能（ΔfG'°）
- 使用方法：组分贡献法（Component Contribution, CC）
- 工具：eQuilibrator API

## 计算条件
- 温度：298 K
- pH：10
- pMg：6
- 离子强度：0.25 M

## 计算结果
- UDP的生成自由能：-2488.78 kJ/mol
- 不确定性：± 4.78 kJ/mol

## 计算说明
- 使用了eQuilibrator API的ComponentContribution方法
- 通过search_compound函数找到UDP化合物（ID: 20, InChI: InChI=1S/C9H14N2O12P2/c12-5-1-2-11(9(15)10-5)8-7(14)6(13)4(22-8)3-21-25(19,20)23-24(16,17)18/h1-2,4,6-8,13-14H,3H2,(H,19,20)(H,10,12,15)(H2,16,17,18)/p-3/t4-,6-,7-,8-/m1/s1)
- 在指定条件下（pH=10, pMg=6, T=298K, I=0.25M）计算了标准转化生成自由能

## 结果解读
在给定的生理条件下（高pH和高pMg），UDP具有非常负的生成自由能，表明其在热力学上不稳定，倾向于分解成更稳定的成分。这种高负值的生成自由能也反映了UDP分子中高能磷酸酐键的存在，这在生物化学反应中具有重要意义。
