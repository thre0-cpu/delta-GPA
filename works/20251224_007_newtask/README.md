## 问题描述

计算以下反应控制的稀薄水溶液中钼酸根与硫代钼酸根的逐步水解与硫取代序列：

- MoS₄²⁻ + H₂O(l) ⇌ MoOS₃²⁻ + H₂S(aq)        K₁ = 1.3 × 10⁻⁵
- MoOS₃²⁻ + H₂O(l) ⇌ MoO₂S₂²⁻ + H₂S(aq)      K₂ = 1.0 × 10⁻⁵
- MoO₂S₂²⁻ + H₂O(l) ⇌ MoO₃S²⁻ + H₂S(aq)      K₃ = 1.6 × 10⁻⁵
- MoO₃S²⁻ + H₂O(l) ⇌ MoO₄²⁻ + H₂S(aq)        K₄ = 6.5 × 10⁻⁶

初始条件：
- [MoS₄²⁻]₀ = 2.00 × 10⁻⁷ mol/L
- pH = 5.00
- V_aq = 1.00 L, V_g = 5.00 L
- T = 298.15 K
- H₂S的亨利常数 H = 9.741 × 10⁻³ atm/(mol·m⁻³)

- H₂S(g)分压 = 2.568×10⁻⁶ atm

### 浓度/气压乘积比值

- **浓度/气压乘积比值: 4.333×10⁻¹³**

## 计算方法

使用数值方法求解包含化学平衡、质量守恒和气液平衡的非线性方程组。通过scipy库的数值求解器确保计算的稳定性和准确性。

关键修正：
1. 正确考虑了每个钼物种产生H₂S当量的差异（S(x)函数）
2. 正确进行了亨利常数的单位转换（从 atm/(mol/m³) 到 atm/(mol/L) 乘以1000）
