# UDP生成自由能计算与离子强度影响分析

## 项目简介

本项目使用eQuilibrator API中的组分贡献法（Component Contribution, CC）计算了UDP（尿苷二磷酸）在特定条件下的生成自由能，并分析了离子强度对其自由能的影响。

## 计算条件

- 温度：298K
- pH：10
- pMg：6
- 离子强度范围：0.1M ~ 0.5M

## 主要结果

### UDP在标准条件下的生成自由能

在温度298K，pH=10，pMg=6条件下：
- UDP的生成自由能：**Δ_fG'° = -1837.63 ± 4.78 kJ/mol**

### 离子强度对UDP生成自由能的影响

随着离子强度的增加，UDP的生成自由能逐渐降低（变得更负）：
- 离子强度 0.100M: -1836.36 ± 4.78 kJ/mol
- 离子强度 0.311M: -1837.96 ± 4.78 kJ/mol
- 离子强度 0.500M: -1838.72 ± 4.78 kJ/mol

在整个离子强度范围内（0.1M到0.5M），UDP的生成自由能变化了约2.36 kJ/mol。

## 结论

UDP在给定条件下具有高度负的生成自由能，表明其生成是一个高度放能的过程。离子强度对UDP的生成自由能有显著影响，随着离子强度增加，生成自由能逐渐降低，这反映了离子强度对生物分子热力学性质的重要影响。
